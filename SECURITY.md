# Security Policy

## Reporting a vulnerability

Please report vulnerabilities privately by contacting the maintainer directly (do not open a public issue with exploit details).

Include:
- Affected version/commit
- Reproduction steps
- Impact assessment
- Suggested remediation (if known)

You will receive acknowledgement as soon as practical, followed by triage and remediation updates.

## Supported versions

Security fixes are prioritized for the latest release line.

| Version | Supported |
|---------|-----------|
| v0.2.x  | ✅ |
| < v0.2  | ❌ |
